Power BI Data Analysis: Indian Takeaway Orders

Time Series Analysis of Two Indian Restaurants

This project analyzes a comprehensive dataset of 19,560 Indian takeaway orders from two restaurants over time, sourced from Kaggle. The analysis provides valuable insights into sales patterns, customer behavior, and menu performance.

Key Features of the Analysis:

  • Time Series Visualization: Tracked sales trends over days, weeks, and months to identify seasonal patterns and growth trajectories
  • Product Performance: Analyzed which menu items were most popular and profitable at each restaurant
  • Customer Behavior: Examined order frequency, average order value, and preferred ordering times
  • Comparative Analysis: Benchmarked performance between the two restaurants to identify operational differences
  • Actionable Insights: Provided data-driven recommendations for menu optimization and promotional timing

Technical Implementation:

The project leveraged Power BI's advanced capabilities including:

  • Data cleaning and transformation using Power Query
  • Creation of custom measures and calculated columns using DAX
  • Interactive time series visualizations with drill-down capabilities
  • Custom tooltips and bookmarks for enhanced user experience
  • Performance optimization techniques for handling large datasets

The full dataset is available on Kaggle for those interested in exploring the raw data.
